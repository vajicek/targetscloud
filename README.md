## Prerequisites

## Use devcontainer

Build dev container.
```bash
make build_devcontainer
```

Run dev container.
```bash
make devcontainer
```

Run **web application** in dev container
```bash
make serve_ng
```

### Web application
Init node
```bash
make init
```

Build **web app**.
```bash
make build_webapp
```

Build **release docker image**.
```bash
make build_release
```

Run **release docker image**.
```bash
make run
```
